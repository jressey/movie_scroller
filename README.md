# Movie Scrollor

Scroll an unending list of popular movies

## Installation

`npm install`  
`bower install`  

## Run in browser

`grunt serve`  
  
Visit in browser @ http://localhost:9000/, please use Chrome

## Running Tests

`grunt test`
